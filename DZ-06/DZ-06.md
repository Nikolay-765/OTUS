# ДЗ № 6
## Тема: Тема: Размещаем свой RPM в своем репозитории
Задание:
1. Создать свой RPM пакет (можно взять свое приложение, либо собрать, например, Apache с определенными опциями).
2. Создать свой репозиторий и разместить там ранее собранный RPM.

## Создаем свой RPM пакет сервера Apache (убираем из сборки модули ldap и authuz-ldap, авторизация пользователей через LDAP)

Устанавливаем недастояющие для выполнения LP пакеты
```
[root@repo /]# yum install -y rpmdevtools rpm-build createrepo yum-utils cmake gcc git
```

Загружаем исходники Apache (httpd) из официального репозитория
```
[root@repo /]# mkdir -p /test/rpm
[root@repo /]# cd /test/rpm
[root@repo rpm]# yumdownloader --source httpd
[root@repo rpm]# ls -al

Включаем репозиторий CRB.\
На всякий случай обновляем httpd-2.4.62-7.el9_7.3.src.rpm\
Устанавливам все пакеты (зависимости), необходимые для сборки Apache
```
[root@repo rpm]# dnf config-manager --set-enabled crb
[root@repo rpm]# rpm -Uvh ./httpd-2.4.62-7.el9_7.3.src.rpm
[root@repo rpm]# yum-builddep httpd
```

## Создаем свой репозиторий и размещаем там свой вариант сборки Apache

Устанавливаем и запускаем Nginx\
Открываем к нему доступ из вне
```
[root@repo /]# dnf install nginx -y
[root@repo /]# systemctl enable --now nginx
[root@repo /]# firewall-cmd --permanent --add-service=http
[root@repo /]# firewall-cmd --permanent --add-service=https
[root@repo /]# firewall-cmd --reload
```

Создаем директорию для репозитория, копируем туда собранные пакеты Apache, догружаем пакеты system-logos-httpd, apr, apr-util\
Инициализируем репозиторий
```
[root@repo /]# mkdir /usr/share/nginx/html/my_repo
[root@repo /]# cp ~/rpmbuild/RPMS/x86_64/*.rpm /usr/share/nginx/html/my_repo/
[root@repo /]# yumdownloader --resolve --destdir=/usr/share/nginx/html/my_repo/ system-logos-httpd
[root@repo /]# yumdownloader --resolve --destdir=/usr/share/nginx/html/my_repo/ apr
[root@repo /]# yumdownloader --resolve --destdir=/usr/share/nginx/html/my_repo/ apr-util
[root@repo /]# createrepo /usr/share/nginx/html/my_repo/
Directory walk started
Directory walk done - 31 packages
Temporary output repo path: /usr/share/nginx/html/my_repo/.repodata/
Preparing sqlite DBs
Pool started (with 5 workers)
Pool finished
```

Добавляем в /etc/nginx/nginx.conf строки:\
    index index.html index.htm\
    autoindex on\
Проверяем корректность настройки WEB сервера и перезагружаем его
```
[root@repo /]# nano /etc/nginx/nginx.conf
[root@repo /]# nginx -t
nginx: the configuration file /etc/nginx/nginx.conf syntax is ok
nginx: configuration file /etc/nginx/nginx.conf test is successful
[root@repo /]# nginx -s reload
```

Создаем файл конфигурации нашего репозитория и прописываем в него:\
    [my_repo]\
    name=myrepo\
    baseurl=http://192.168.0.150/my_repo \
    pgpcheck=0\
    enabled=1
```
