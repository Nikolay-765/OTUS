# ДЗ № 3
## Тема: Работа с LVM
Задание
1. Уменьшить том под / до 8G.
2. Выделить том под /home.
3. Выделить том под /var - сделать в mirror.
4. /home - сделать том для снапшотов.
5. Прописать монтирование в fstab.
6. Работа со снапшотами:
    + сгенерить файлы в /home/;
    + снять снапшот;
    + удалить часть файлов;
    + восстановиться со снапшота.

## Готовим LVM под все ДЗ.
При этом:
 - под LV для home выделяем 70% VG (30% оставляем под shapshot)
 - для var делаем LV с зеркалированием
```
root@server:/home/user# pvcreate /dev/sd{b..e}
root@server:/home/user# vgcreate vg_root /dev/sdb
root@server:/home/user# vgcreate vg_home /dev/sdc
root@server:/home/user# vgcreate vg_var /dev/sdd /dev/sde
root@server:/home/user# lvcreate -n lv_root -l +100%free /dev/vg_root
root@server:/home/user# lvcreate -n lv_home -l +70%free /dev/vg_home
root@server:/home/user# lvcreate -n lv_var -m1 -l +100%free /dev/vg_var
```

## Уменьшаем LV с root`ом до 8 Gb
Создаем на lv_root файловую систему.
Монтирует lv_root в /mnt/root_tmp
Копируем root в /mnt/root_tmp (в пределах ФС на ubuntu--lv) 
Монтируем /boot и виртуальные ФС в соответствующие директории в /mnt/root_tmp/
```
root@server:/home/user# mkfs.ext4 /dev/vg_root/lv_root
root@server:/home/user# mkdir /mnt/root_tmp
root@server:/home/user# mount /dev/vg_root/lv_root /mnt/root_tmp
root@server:/home/user# rsync -axHAX --progress / /mnt/root_tmp/
root@server:/home/user# for i in /proc /sys /dev /run /boot; do mount --bind  $i /mnt/root_tmp$i; done
```

Переносим root на lv_root (на /dev/sdb)
```
root@server:/home/user# chroot /mnt/root_tmp/
root@server:/# grub-kmconfig -o /boot/grub/grub.cfg
root@server:/# update-initramfs -u
```

Перезагружаемся и уменьшаем размер ubuntu-lv до 8 Gb
```
root@server:/home/user# e2fsck -fy /dev/ubuntu-vg/ubuntu-lv
root@server:/home/user# resize2fs /dev/ubuntu-vg/ubuntu-lv 8G
root@server:/home/user# lvreduce /dev/ubuntu-vg/ubuntu-lv -L 8G
```

Монтируем ubuntu-lv в /mnt/root_tmp/ и смотрим, что получилось.

## Перекидываем /var в lv_var

Создаем ФС на lv_var и монтируем ее в /mnt/var/
Копируем содержимое /var на lv_var и для постраховки вереносим все из /var в /old_var
Монтируем lv_var в /var
```
root@server:/home/user# mkfs.ext4 /dev/vg_var/lv_var
root@server:/home/user# mkdir /mnt/var
root@server:/home/user# mount /dev/vg_var/lv_var /mnt/var/
root@server:/home/user# cp -aR /var/* /mnt/var/
root@server:/home/user# mkdir /old_var
root@server:/home/user# mv /var/* /old_var
root@server:/home/user# umount /mnt/var/
root@server:/home/user# mount /dev/vg_var/lv_var /var
```

Выясняем UUID /dev/vg_var/lv_var, использую lsblk -f (но для скрипта удобнее blkid)
Добавляем в /etc/fstab следующую строку c UUID /dev/vg_var/lv_var
```
UUID=7b45af31-71a3-4db4-8cdb-f0681ffea9f6 /var ext4 defaults 0 2
```

Перезагружаемся и проверяем, что вышло

## Перекидываем /home на lv_home и работаем со snapshot`ом

```
root@server:/home/user# mount /dev/vg_home/lv_home /mnt/home/
root@server:/home/user# cd /

root@server:/# umount /mnt/home/
root@server:/# mount /dev/vg_home/lv_home /home
```
